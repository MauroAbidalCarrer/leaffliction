# Leaffliction
*Classification of leafs images with a simple CNN.*

## Installation
> If you are a 42 school student constrained by your session storage you may want to execute the following commands:
> ```shell
> mkdir -p ~/sgoinfre/venvs/leaffliction
> # Tells uv to install the packages in sgoinfre 
> export UV_PROJECT_ENVIRONMENT=$HOME/sgoinfre/venvs/leaffliction instead of the repo
> # Tells uv to not print warnings that I ... don't understand
> export UV_LINK_MODE=copy 
> ``` 
```shell
git clone https://github.com/MauroAbidalCarrer/leaffliction.git
cd leaffliction
uv sync
```
